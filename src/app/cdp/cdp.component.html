Capstone Design Project

The task of this project is to create an Augmented Reality application that will help people navigate through a building using visual cues. As mentioned, this app will rely on Augmented Reality (AR) technology; a technology that is intimately tied with your location and that can access information sources that will improve your understanding of how to reach your destination. This app should have a user-friendly Graphical User Interface (GUI) with a scroll or drop down menu for all the rooms in the building.

The idea of the app is to build an interface in android studio that allows us to select the information required, then send that information to a unity project that utilizes the camera to perform image recognition through vuforia to generate directional arrows for the user to follow on their journey through the building.
